Polygon Feature: Country Cut-Out of Switzerland¶

The following notebook demonstrates the full workflow to access, process and visualize ICON-CH1-EPS model data within the border of Switzerland using 2-meter temperature data.

The data is retrieved using Polytope, a feature extraction software developed by ECMWF. It applies concepts of computational geometry to extract n-dimensional polygons (also known as polytopes) from datacubes, such as a country cut-out. Follow the instructions to learn more about model data access via Polytope.

Installation¶

Follow the instructions in README.md to install the necessary dependencies.

Configuring Access to Polytope¶

To access ICON data via MeteoSwiss's Polytope, you need a Polytope offline token provided by MeteoSwiss. If you do not already have a token, you can request one here. Then, create a new config.yml file based on config_example.yml, and replace <meteoswiss_key> with your access token there.

In [1]:
import os
import yaml

def load_config(path="config.yml"):
    if not os.path.exists(path):
        raise FileNotFoundError("Missing config.yml. Please create one based on config_example.yml.")
    with open(path, "r") as f:
        return yaml.safe_load(f)

config = load_config()

#ICON-CSCS Polytope credentials
os.environ["POLYTOPE_USER_KEY"] = config["meteoswiss"]["key"]
os.environ["POLYTOPE_ADDRESS"] = "https://polytope-depl.mchml.cscs.ch"

Selecting date and time of the forecast¶

The FDB containing real-time data typically holds only the most recent day of forecasts. Therefore, it is necessary to specify the current date and select a corresponding forecast time in the past.

In [2]:
from datetime import datetime, timedelta

# Current time
now = datetime.now()

# Subtract 12 hours
past_time = now - timedelta(hours=12)

# Round down to the nearest multiple of 6
rounded_hour = (past_time.hour // 6) * 6
rounded_time = past_time.replace(hour=rounded_hour, minute=0, second=0, microsecond=0)

# Format as YYYYMMDD and HHMM
date = rounded_time.strftime('%Y%m%d')
time = rounded_time.strftime('%H%M')
date,time

Extract the country polygon of Switzerland¶

In our example we are looking for the shape of Switzerland with a resolution of 50'000'000 meters. With earthkit.geo it is possible to extract one or more country shapes using the function country_polygons().

In [3]:
import earthkit.geo.cartography

shapes = earthkit.geo.cartography.country_polygons("Switzerland", resolution=50e6)
points_lat_lon = shapes[0]

Define the request¶

We need to define a MARS request. The feature attribute allows you to extract only the relevant data at the given points. Thus, the amount of data that is retrieved from storage is significantly reduced. For the "polygon" feature the following dictionary is needed.

NOTE: Don't forget to specify that the data points are tuples that first contain latitude and then longitude.

In [4]:
feature={
    "type" : "polygon",
    "shape" : points_lat_lon,
    "axes" : ["latitude", "longitude"] # first latitude, then longitude
}

Finally, we can define the request. This example fetches 2-m temperature from ICON-CH1-EPS at the surface, for ensemble member 1 (number=1), at the selected run date/time.

  • type="pf" — perturbed member (requires number).
  • type="cf" — control forecast (no number).
In [5]:
request = {
    'param': '500011',
    'date': date,
    'time': time,
    'expver': '0001',
    'number': 1,
    'step': 0,
    'class': 'od',
    'levtype': 'sfc',
    'model': 'icon-ch1-eps',
    'stream': 'enfo',
    'type': 'pf',
    'feature': feature,
    'timespan': 'none'
}

Data retrieval¶

Now we use earthkit.data to load the data and convert it into an xarray.Dataset.

ds = ekd.from_source(

    "mchgj",
    request,
    stream=False

Plotting¶

We use the library earthkit.plots to plot the data. Moreover, we can specify a styling with earthkit.plots.styles to show a color spectrum from red to blue and convert the unit to degrees Celsius for better readability.

In [7]:
from earthkit.plots import Map
from earthkit.plots.styles import Style
from earthkit.plots.styles.levels import Levels
from earthkit.plots.geo import bounds, domains
import cartopy.crs as ccrs
import matplotlib.ticker as mticker

xmin, xmax = 5, 11   # Longitude bounds
ymin, ymax = 45.5, 48   # Latitude bounds

bbox = bounds.BoundingBox(xmin, xmax, ymin, ymax, ccrs.Geodetic())
domain = domains.Domain.from_bbox(bbox=bbox, name="CH2")

style = Style(
    colors="RdBu_r",
